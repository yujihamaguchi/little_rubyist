# /refactor コマンド

既存コードのリファクタリングを行う。

## 使い方

```
/refactor [ファイルパスまたはディレクトリ]
```

## 例

```
/refactor lib/state/
/refactor lib/little_rubyist.rb
/refactor test/mediator/mediator_e2e_test.rb
```

## 動作

1. 対象コードを読み込み、現状を把握
2. テストがパスすることを確認
3. リファクタリングを実施
4. テストで動作確認

## リファクタリングの観点

- コードの重複排除
- メソッドの適切な分割
- 命名の改善
- 不要コードの削除

## 注意事項

- テストがパスする状態を維持
- 一度に大きな変更をしない
- リファクタリングと機能追加を混ぜない
